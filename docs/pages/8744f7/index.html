# C++ Preprocess（预处理）

一些预处理指令, 由C++ 预处理器（preprocessor (opens new window)）进行处理；

if (opens new window) ifdef (opens new window) include (opens new window) defined (opens new window) export (opens new window) (C++20)
elif (opens new window) ifndef (opens new window) line (opens new window) __has_include (opens new window) (since C++17) import (opens new window) (C++20)
else (opens new window) define (opens new window) error (opens new window) __has_cpp_attribute (opens new window) (since C++20) module (opens new window) (C++20)
endif (opens new window) undef (opens new window) pragma (opens new window)

following tokens are recognized by the preprocessor (opens new window) outside the context of a preprocessor directive:

  • _Pragma (opens new window) (since C++11)

  1. # conditionally ：#if (opens new window) #else (opens new window) #elif (opens new window) #ifdef (opens new window) #ifndef (opens new window) #endif (opens new window) #elifdef, #elifndef (since C++23)

  2. # replace：

    • #define (opens new window) #undef (opens new window)
    • # : 在运行形参替换的基础上以引号包围，实际上创建一个字符串字面量。
    • ##: 在运行形参替换的基础上将结果进行拼接。
    • 其他的一些Macro:
      • _FILE_ 展开成当前文件名，作为字符串字面量，可用 #line 指令更改(宏常量)
      • __LINE: 展开成源文件行号，整数常量，可用 #line 指令更改(宏常量)
      • __DATE: 展开成翻译日期，形式为 "Mmm dd yyyy" 的字符串。如果月中日期数小于 10 则 "dd" 的首字符为空格。月份名如同以 std::asctime() 生成(宏常量)
      • _TIME_: 展开成翻译时间，形式为 "hh:mm:ss" 的字符串字面量(宏常量)

  3. # Include

    1. #include (opens new window)
    2. __has_include (opens new window) (C++17 起) 提供的可以判断是否include这个h文件
    3. __has_cpp_attribute (opens new window) (C++20 起)

  4. # error

    1. #error (opens new window)

  5. # Implementation defined : behavior control

    1. #pragma (opens new window)
    2. _Pragma (opens new window)(C++11 起)
    3. #pragma STDC
    4. #pragma once: 编译器只对其分析一次，即使它在同一源文件中（直接或间接）被包含了多次也是如此。

    类似的标准方式:

#ifndef LIBRARY_FILENAME_H
#define LIBRARY_FILENAME_H
// 头文件的内容
#endif /* LIBRARY_FILENAME_H */
#pragma pack: 控制后续定义的类和union的最大对齐方式;
